# User Guide

## Getting Started
1. Install Python
2. Run main.py

## Main Menu Options
1. Add Expense
2. View All Expenses
3. Generate Report
4. Export Data
5. Exit

## Adding an Expense
- Enter date
- Choose category
- Enter amount
- Add description

## Viewing Reports
- Monthly summary
- Category-wise totals

## Data Storage
- Data saved automatically in expenses.csv

## Common Errors
- Invalid date format
- Non-numeric amount
