# Reports Folder

This folder contains auto-generated output files created by the
Personal Finance Manager application.

## Files Included

- category_summary.txt
  Category-wise expense totals

- monthly_report.txt
  Monthly expense statistics

- category_expenses.png
  Bar chart of expenses by category

- monthly_trend.png
  Line chart showing monthly spending trend

These files are generated automatically when the user runs
report or visualization features in the application.
